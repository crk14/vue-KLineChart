# vue_klineChart

klinechart 是一个轻量级的K线图插件（基于html5 canvas构建，图表支持20种技术指标，支持自定义指标参数，支持绘制标记图形，支持自定义样式） https://github.com/liihuu/KLineChart

相比tradingview 更适合小白

```
npm install
```
